# Судоку

Графическая игра Судоку с фиксированными уровнями и статистикой.

## Как запустить

```bash
python sudoku.py
```

## Геймплей

- **Меню уровней** — при запуске показывается список из 10 карт со сложностью и лучшим временем
- **Левый клик** по пустой ячейке → открывается панель с цифрами
- **Левый клик** по неверной цифре (красная) → открывается панель для замены
- **Левый клик** по правильной цифре (зелёная/исходная) → игнорируется
- **Повторный клик** по той же ячейке → панель закрывается
- **Клик по другой ячейке** при открытой панели → старая панель закрывается, новая открывается у курсора
- **Клик вне поля или по кнопкам** → панель закрывается
- **Правый клик** по пустой ячейке открывает панель заметок (карандаш)
- **Таймер** запускается при старте уровня; при победе время сохраняется, если оно лучше предыдущего рекорда

## Горячие клавиши

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+Z` | Откат хода |
| `Ctrl+Y` | Возврат отменённого хода |

## Интерфейс

| Элемент | Описание |
|---------|----------|
| Меню | Список карт с названием, цветной меткой сложности и лучшим временем |
| Верхняя панель | Название карты, значок сложности, таймер |
| Игровое поле | Сетка 9×9 |
| Панель цифр | Всплывающее окно с кнопками 1–9 |
| Нижняя панель | Кнопки «Отменить», «Повторить», «К выбору уровней» |

## Визуальные обозначения

- **Серый фон** — исходные цифры (даны в уровне, нельзя менять)
- **Красный фон** — неверно поставленная цифра
- **Зелёный фон строки/столбца/блока** — область полностью и верно заполнена
- **Голубой фон** — выбранная ячейка
- **Зелёный текст** — верно поставленная игроком цифра
- **Красный текст** — неверная цифра
- **Серые цифры в углах** — заметки (карандаш)

## Уровни сложности

- 🟢 **Легкий** — 4 карты (36–42 подсказки)
- 🟠 **Средний** — 4 карты (30–35 подсказок)
- 🔴 **Сложный** — 2 карты (26–28 подсказок)

## Файлы

- `sudoku.py` — игра
- `sudoku_stats.json` — статистика (лучшее время по каждой карте, создаётся автоматически)
